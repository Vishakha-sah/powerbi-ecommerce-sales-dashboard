# 📊 Power BI E-Commerce Sales Dashboard

## Overview

This project is an interactive **Power BI dashboard** built using an e-commerce sales dataset. The dashboard provides insights into sales performance, profit, customer purchases, payment modes, and product categories using interactive visualizations and filters.

This project was created by following a Power BI tutorial as a learning exercise. While building it, I explored Power BI features such as data visualization, formatting, slicers, cross-filtering, and dashboard design.

---

## Dashboard Features

* KPI Cards for:

  * Total Sales
  * Total Profit
  * Total Quantity
  * Average Order Value (AOV)

* Interactive Charts:

  * Total Sales by State
  * Profit by Month
  * Total Profit by Sub-Category
  * Total Sales by Customer
  * Quantity by Payment Mode
  * Quantity by Category

* Interactive Filters:

  * Quarter
  * State

* Cross-filtering and cross-highlighting between visuals.
